== Description ==

HappenStance is an easily customizable theme which can be used for your Blog, Magazine, Business, eCommerce or Events website. It is a fully responsive and Retina ready theme that allows for easy viewing on any device.

== Installation ==

You can install this theme through the WordPress installer under "Appearance > Themes - Add New" by searching for it.
Alternatively, you can download the file, unzip it and move the extracted content via FTP to the "wp-content/themes" location on your web server. Then, you will be able to activate the theme in "Appearance > Themes".

== Documentation ==

The complete documentation of HappenStance names "documentation.html" and is located in "docs" folder in this theme main folder. It is also accessable via Theme Options Panel after installing and activating this theme.
